## [0.1.0] - 2026-02-07

### Added

- USGS earthquake API client with configurable parameters
- Earthquake and EarthquakeCatalog data models with GeoJSON parsing
- Statistical analysis: Gutenberg-Richter law, b-value, anomaly detection
- Alert system with configurable threshold rules
- Interactive Folium map generation
- Matplotlib plotting for magnitude-time and G-R distributions
- CLI with fetch, analyze, map, and monitor commands
- Unit, integration, and end-to-end test suites
- GitHub Actions CI/CD pipelines
- Sphinx documentation with ReadTheDocs support
