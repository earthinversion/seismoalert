Installation
============

Requirements
------------

- Python 3.11 or later

Install from PyPI
-----------------

.. code-block:: bash

   pip install seismoalert

Install from Source
-------------------

.. code-block:: bash

   git clone https://github.com/utpalkumar/seismoalert.git
   cd seismoalert
   pip install -e .

Development Installation
------------------------

To install with development and testing dependencies:

.. code-block:: bash

   pip install -e ".[dev,test,docs]"
